# DIAPOSITIVA 5
## Caso inicial: Agricultura

### Problema actual:
Los productores reaccionan cuando el problema ya es visible.

### Nuestro enfoque:
Detección temprana basada en:
* NDVI / NDWI
* Datos climáticos
* Históricos de cultivo
* Feedback de campo

**Resultado:** Intervención más rápida, menos pérdidas y mayor productividad.

# DIAPOSITIVA 8
## Modelo de negocio

* **Nivel 1 (Piloto de Validación):** Setup inicial. Duración 30-45 días. Objetivo: Demostrar ROI.
* **Nivel 2 (Suscripción Mensual):** Cobro recurrente por hectárea, activo o instalación monitoreada.
* **Nivel 3 (Enterprise):** Licencias anuales, acceso a APIs, modelos personalizados e integraciones profundas.

---

# DIAPOSITIVA 9
## Expansión sectorial

* **Agricultura:** Detectar problemas antes de que afecten el rendimiento físico de la producción.
* **Seguros:** Evaluación automática de riesgos, validación remota de siniestros y monitoreo de eventos.
* **Infraestructura:** Monitoreo continuo de activos, detección de cambios estructurales y mantenimiento predictivo.
* **Defensa y Seguridad:** Observación territorial autónoma, detección de anomalías y protección de infraestructura crítica.

# DIAPOSITIVA 10
## Nuestro verdadero activo

### El Loop de Retroalimentación:
Cada alerta genera feedback. El usuario confirma en tiempo real:
* ¿Fue un problema real?
* ¿Fue una falsa alarma?
* ¿Qué acción tomó?

Cada interacción alimenta la plataforma. Con el tiempo construimos **datasets propietarios y modelos más precisos**. La plataforma mejora con cada cliente.
